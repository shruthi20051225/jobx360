## 🚀 GitHub Pages Deployment
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/jobx360.git
git push -u origin main
```
Then: Repo → Settings → Pages → Source: main → Custom domain: your-domain.com
